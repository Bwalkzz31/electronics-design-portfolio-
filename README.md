## Projects

### 1. OR Gate Built from NAND Gates

![OR Gate using NAND](nand-or-gate.png)

**Description:**
Designed and analyzed an OR logic gate constructed entirely from NAND gates using Boolean algebra transformation.

**Logic Expression:**
A + B = (A NAND A) NAND (B NAND B)

**Key Concepts Demonstrated:**
- Functional completeness of NAND gates
- Logic transformation and equivalence
- Gate-level circuit design

**Outcome:**
Successfully showed how complex logic functions can be implemented using a single universal gate type.
